## ✨ 關於 GOODTOGO
                                                                                           
減少剩食以愛護地球為理念所創建的平台，提供商家販售即將到期的食品、申請為
會員的使用者於網站上購買即期食物，並藉由網站傳達惜食的觀念，幫助降低因人
類產生過多的碳排放，期許將良好的用意、理念，透過平台讓更多人一起加入行
動。                                                                                           

## 🥳 使用方式

```bash
$ git clone https://github.com/dosskkr/GOODTOGO.git
$ cd GOODTOGO
```

```bash
$ cd be
$ npm i
$ npm run dev
```

```bash
$ cd fe
$ npm i
$ npm start
```
